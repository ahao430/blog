

安装claude code

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

再配置环境变量

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc
```



接下来配置第三方模型

ccswitch cli: [https://github.com/SaladDay/cc-switch-cli/blob/main/README_ZH.md](https://github.com/SaladDay/cc-switch-cli/blob/main/README_ZH.md)

安装命令

```bash
curl -fsSL https://github.com/SaladDay/cc-switch-cli/releases/latest/download/install.sh | bash
```

使用命令

```bash
cc-switch
```
