【开源】cc-touchbar

发表于2026-06-28|更新于2026-06-28|开源项目
|浏览量:

GitHub - ahao430/cc-touchbar: claude code的touchbar增强，在touchbar显示claude code模型、时间、用量、余额等信息。

自用macbook m1。这两天想要好好利用一下touchbar，想到用来展示claude code编程时候的一些模型和会话信息正合适。用AI实现了一个，支持m芯片和intel芯片，支持touchbar的机型。

如图，展示了当前的: 供应商、模型|剩余额度|上下文、会话token、缓存占比|两个快捷启动按钮。
